If you see an error like `'vite' is not recognized`, it means you need to install the software dependencies first.

### Step 1: Install Dependencies
Open your terminal in this folder and run:
```bash
npm install
```
*This downloads the required libraries to your computer. You only need to do this once.*

### Step 2: Run the App
Once installation is complete, run:
```bash
npm run dev
```
*Then click the link (usually http://localhost:5173) to open the software.*
